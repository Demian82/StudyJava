Java 과제
컬렉션 프레임워크, 제네릭, 클래스를 이용한 도서 관리 프로그램 구현

도서 등록/삭제/조회
-도서 정보(도서 ID, 제목, 저자, 연도 등)
-기능 (도서 등록, 도서 삭제, 전체 도서 출력, 검색 등)

회원(Member)등록/삭제/조회
-회원 정보 (회원 ID, 이름, 연락처 등)
-기능 (새 회원 등록, 회원 삭제, 회원 목록 출력)

대출/반납 기능
-규칙 등 (한 회원은 여러 도서를 대출 가능, 도서 1권은 동시에 한 명에게만 대출)

정렬 및 통계 기능
-정렬 기능(제목 오름차순, 출판연도 내림차순 등)
-통계 기능(현재 대출 중인 도서 수 등)
